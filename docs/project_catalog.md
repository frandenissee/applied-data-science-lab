# Catálogo de Proyectos

## Propósito

Este documento reúne los proyectos candidatos del **Applied Data Science Lab**.

El objetivo es identificar problemas reales cuya resolución requiera la integración de matemáticas, estadística, ciencias de la computación e ingeniería para construir soluciones basadas en datos.

Cada proyecto representa una oportunidad para fortalecer competencias desarrolladas durante la Licenciatura en Ingeniería en Ciencia de Datos y contribuir a un portafolio académico-profesional sólido.

## Filosofía del catálogo

En este laboratorio los proyectos no nacen a partir de algoritmos ni de datasets.

Cada proyecto comienza con un problema real.

A partir de ese problema se identifican las preguntas relevantes, se estudia el dominio, se seleccionan las herramientas adecuadas y finalmente se desarrolla una solución basada en datos.

## Criterios de selección

Todo proyecto candidato será evaluado considerando los siguientes aspectos:

- Relevancia del problema.
- Impacto potencial.
- Relación con la formación académica.
- Valor de aprendizaje.
- Disponibilidad de datos.
- Nivel de desafío técnico.
- Valor para el portafolio profesional.
- Posibilidad de extender el proyecto en el futuro.
